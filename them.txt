hướng dẫn: 
- import database hoặc dán câu lệnh tạo bảng và chạy trên sql server 
- mở project bằng visual code đã cài asp.net và mở bằng cách mở file Program_WEB_BV > .sln
- sửa file web.config: 
	+ data source=<tên server của bạn trong sql server> 
	+ initial catalog=<tên database bạn đặt hoặc k đặt thì không sửa phần này vì đã cài mặc định>
	+ user id=<username đăng nhập sql server của bạn>
	+ password=<password đăng nhập sql server của bạn>
